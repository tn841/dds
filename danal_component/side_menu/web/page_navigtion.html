Pagination

페이지네이션은 페이지를 나누고, 이전 혹은 다음 페이지로 넘어가거나 특정 페이지로 넘어갈 수 있는 일련의 링크를 제공합니다.

Demo

1 2 3 4 5

현재 보고 있는 페이지는 클래스를 표시하여 사용한다. 현재 위치를 나타낼 때, a에 스타일을 추가하기 위해 class="on"를 표시한다.

태그의 스타일을 지정할 때에 class="사이즈, 스타일" 순서를 지키도록 한다.

<div class="dn_pag_md">
    <a class="first" href="#"></a>
    <a class="pre" href="#"></a>
    <a class="on" href="#">1</a>
    <a href="#">2</a>
    <a href="#">3</a>
    <a href="#">4</a>
    <a href="#">5</a>
    <a class="next" href="#"></a>
    <a class="last" href="#"></a>
</div>

Pagination Sizes

페이지네이션의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
페이지네이션의 사이즈는 dn_pag_sm, dn_pag_md, dn_pag_lg 으로 구성 되어있습니다.
사이즈 클래스만 사용할 경우 Default 스타일로 버튼 스타일이 적용됩니다.

Pagination Styles

페이지네이션의 스타일은 총 4가지로 분류되어 있습니다.
기본 페이지네이션 사이즈만 사용할 경우를 Default 스타일로, class="style01", class="style02", class="style03" 를 추가 사용할 경우 총 4가지의 스타일을 나타낼 수 있습니다.
