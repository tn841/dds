DTD(Document Type Definition)

HTML 문서 첫줄에 !DOCTYPE 로 시작하는 문법을 이야기 하며 작성된 내용이 HTML 표준에 의거하여 작성되었음을 웹 브라우져에게 알려주는 것, W3C에서 제공하는 HTML 표준에 따라 보여주도록 권고하기 위함입니다. <!DOCTYPE html>  가장 최신의 웹표준을 지원하는 HTML5 DOCTYPE을 권장

메타테크 viewport

모바일 환경을 위한 태그, 모바일 브라우져들은 viewport 가상의 창에 페이지를 렌더링 하는데 뷰포트의 크기와 스케일을 조정할 수 있고 , 확대 여부기능 조정을 할 수 있습니다. <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no,maximum-scale=1.0, minimum-scale=1.0">

메타테그 “content=”IE=edge”

IE브라우져에서, 각 버전 중 최신 표준 모드를 선택하는 문서 모드입니다.(IE6부터 IE11까지) 문서 유형(documment typ)선언과 함께 사용해야 유효합니다. <meta http-equiv="X-UA-Compatible" content="IE=edge">

C. 영문 소문자, 숫자, 언더스코어 조합
- 영문 소문자, 숫자, 언더스코어(_)만 사용할 수 있다.
- 언더스코어(_)는 단어와 단어 조합할 때만 사용한다.
- 언더스코어(_)가 포함된 약속어는 숫자, 영문 소문자와 조합하여 사용할 수 있다.
