Table

테이블(Table)이란 여러 종류의 데이터를 정리하여 보여주는 표를 의미합니다.

Demo

테이블의 th, td 행간(Column) 조절은 colgroup 내에 col width="#" 를 사용하여 사이즈를 지정한다.

테이블 태그의 스타일을 지정할 때에 class="사이즈, 스타일" 순서를 지키도록 한다.
