Tabs

탭은 사용자가 동일한 컨텐츠 안에서 쉽게 탐색할 수 있도록 몇 가지 관련된 섹션으로 구분하여 사용합니다.

탭 태그는 ul, li를 주로 사용한다.

li의 width 를 조정하여 갯수와 넓이를 정하여 사용한다.

“사용되지 않는 링크” 나 “현재 보고 있는 페이지 등은 클래스를 표시하여 사용한다. 현재 위치를 나타낼 때, li에class="on"코드를 사용한다.

Tabs Sizes

탭의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
탭 태그의 사이즈는 dn_tabs_sm, dn_tabs_md, dn_tabs_lg 으로 구성 되어있습니다.
사이즈 클래스만 사용할 경우 Default 스타일로 버튼 스타일이 적용됩니다.

Tabs Style

탭 스타일은 총 4가지로 분류되어 있습니다.
기본 탭 사이즈만 사용할 경우를 Default 스타일로, class="style01", class="style02", class="style03" 를 추가 사용할 경우 총 4가지의 스타일을 나타낼 수 있습니다.
