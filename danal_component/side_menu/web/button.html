Button

버튼은 사용자가 데이터를 전송하거나 기능을 실행하게 하는 어떠한 행동을 유도합니다. 다양한 필요를 위해 여러 스타일이 제공됩니다.

Demo

버튼 태그는 button, input type="button", a 3가지를 주로 사용한다.

button, input 은 type="button" 을 필수로 뒤에 덧붙여 사용한다.

a 는 role="button" 을 필수로 뒤에 덧붙여 사용한다.

1
2
3
<button type="button" class="dn_btn_lg">Button</button> <!--button 태그-->
<input type="button" value="Button" class="dn_btn_lg"> <!--input 태그--> 
<a href="#" role="button" class="dn_btn_lg">Button</a> <!--a 태그-->
cs

버튼 태그의 스타일을 지정할 때에 class="사이즈, 스타일" 순서를 지키도록 한다.

1
2
<button type="button" class="dn_btn_lg btn_secondary">.btn_secondary</button>
<button type="button" class="dn_btn_lg btn_line">.btn_line</button>
cs

Button Sizes

버튼의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
버튼 태그의 사이즈는 dn_btn_sm, dn_btn_md, dn_btn_lg 으로 구성 되어있습니다.
사이즈 클래스만 사용할 경우 Default 스타일로 버튼 스타일이 적용됩니다.

Button Styles

버튼 스타일

버튼의 스타일은 총 3가지로 분류되어 있습니다.
기본 버튼 사이즈만 사용할 경우를 Default 스타일로 class="btn_secondary", class="btn_line" 를 추가 사용할 경우 총 3가지의 스타일을 나타낼 수 있습니다.

1
2
3
<button type="button" class="dn_btn_lg">Default</button>
<button type="button" class="dn_btn_lg btn_secondary">.btn_secondary</button>
<button type="button" class="dn_btn_lg btn_line">.btn_line</button>

버튼 태그과 앵커 요소는 비활성 사용법이 다르니 하단의 예문을 확인해주세요.
