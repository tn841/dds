Checkbox

체크박스는 옵션 목록에서 하나 혹은 여러개의 옵션을 선택하도록 합니다.

Demo

체크박스가 텍스트와 함께 그룹으로 선택되기를 원할 때 input type="checkbox" id="check01", label for="check01" input 태그 id 와 label 태그 for 값을 동일하게 한다.

체크박스 태그의 스타일을 지정할 때에 class="사이즈, 스타일" 순서를 지키도록 한다.

1
2
3
4
5
6
<form class="chk_inline">
    <div class="dn_chk_lg chk_solid">
        <input type="checkbox" id="check" title="체크박스 샘플">
        <label for="check">checkbox</label>
    </div>
</form>

Checkbox Sizes

체크박스의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
체크박스의 사이즈는 input을 감싸고 있는 부모 요소에 dn_chk_sm, dn_chk_md, dn_chk_lg 클래스를 추가 해주세요.
사이즈 클래스만 사용할 경우 Default 스타일로 체크박스 스타일이 적용됩니다.
