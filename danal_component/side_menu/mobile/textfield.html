Text_field

텍스트 필드는 사용자가 텍스트를 작성하거나 편집할 수 있는 입력창 입니다.

Demo

텍스트 필드 태그는 input type="text" 를 사용한다.

텍스트 필드 태그의 스타일을 지정할 때에 class="사이즈" 순서를 지키도록 한다.

모바일에서 텍스트 필드 태그의 width는 디폴트 값이 100% 이다.

1
<input type="text" class="dn_text_lg" placeholder="텍스트를 입력하세요">

Text_field Sizes

텍스트 필드 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
모바일의 경우 width는 100%로 정해져있어 높이 값을 기준으로 사이즈 구분이 되어있습니다.
텍스트 필드의 사이즈는 dn_text_sm, dn_text_md, dn_text_lg 로 구성 되어있습니다.

Small 사이즈

1
2
3
<input type="text" class="dn_text_sm" placeholder="텍스트를 입력하세요">
<input type="text" class="dn_text_sm" value=".dn_text_sm">
<input type="text" class="dn_text_sm" value=".dn_text_sm" disabled>

1
2
3
<input type="text" class="dn_text_md" placeholder="텍스트를 입력하세요">
<input type="text" class="dn_text_md" value=".dn_text_md">
<input type="text" class="dn_text_md" value=".dn_text_md" disabled>

<input type="text" class="dn_text_lg" value=".dn_text_lg" disabled>

Text_field Styles

스타일은 총 2가지로 분류되어 있습니다.
기존 사이즈 클래스만 사용할 경우 하단의 스타일로 적용됩니다.

1
<input type="text" class="dn_text_md" placeholder="텍스트를 입력하세요">

Text_field_fill 스타일

class="textfield_fill" 클래스를 추가하면 Select 하단에만 라인이 그려진 스타일로 적용됩니다. 원하시는 스타일로 적용해주세요.

1
<input type="text" class="dn_text_md textfield_fill" placeholder="텍스트를 입력하세요">

Text_field_datepicker 스타일

텍스트 필드에 버튼이 함께 하는 경우가 많습니다. (ex. 텍스트필드+검색버튼) 그럴 경우 하단의 코드를 이용하여 스타일을 입혀주세요. 데이트피커(Datepicker)의 경우 코드를 따로 정리 하였으니 참고해주세요.

1
2
3
4
<div class="dn_input_group">
    <input type="text" class="dn_text_md">
    <button type="button" class="dn_btn_md"><i class="dn_icon_md "i_search"></i></button>
</div>
