Button

버튼은 사용자가 데이터를 전송하거나 기능을 실행하게 하는 어떠한 행동을 유도합니다.
다양한 필요을 위해 여러 스타일이 제공됩니다.

Demo

버튼 태그는 button,input type="button",a 3가지를 사용한다.

button,input은type="button"을 필수로 뒤에 덧붙여 사용한다.

a 는 role="button" 을 필수로 뒤에 덧붙여 사용한다.

1
2
3
<button type="button" class="dn_btn_lg">Button</button><!--button 태그-->
<input type="button" value="Button" class="dn_btn_lg"><!--input 태그-->
<a href="#" class="dn_btn_lg">Button</a><!--a 태그-->
cs

버튼 태그의 스타일을 지정할 때에 class="사이즈,스타일" 순서를 지키도록 한다.

1
2
<button type="button" class="dn_btn_lg btn_secondary">button</button>
<button type="button" class="dn_btn_md btn_line">button</button>
cs

Button Sizes

버튼의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.

Button Style

버튼 스타일은 총 3가지로 분류되어 있습니다.
기본 버튼 사이즈만 적용한 경우를 Default 스타일로 Default, class="btn_secondary", class="btn_line"로 구성되어 있습니다.

Default

Button Disabled

버튼을 비활성화 시키려면 disabled 를 추가하세요
모든 비활성화 버튼의 스타일은 동일합니다.
