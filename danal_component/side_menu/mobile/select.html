Select

Select는 드롭다운 목록을 만들어 그 중 옵션을 선택할 수 있도록 하는 태그입니다.

Demo

select 태그는 옵션 메뉴들을 제공한다. option 태그들은 그룹화할 수 있는 optgroup 태그로 그룹화한다.

Select 태그의 스타일을 지정할 때에 class="사이즈" 를 지키도록 한다.

모바일에서 Select 태그의 width는 디폴트 값이 100% 이다.

        <option>계좌이체</option>
        <option>무통장입금</option>
    </optgroup>
</select>

Select Sizes

모바일 Select의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
모바일의 경우 width는 100%로 정해져있어 높이 값을 기준으로 사이즈 구분이 되어있습니다.
Select의 사이즈는 dn_select_sm, dn_select_md, dn_select_lg 로 구성 되어있습니다.

Select_fill 스타일
