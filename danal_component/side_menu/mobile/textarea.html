Textarea

텍스트에어리어는 여러줄에 걸쳐 많은 내용을 입력할 수 있는 폼 필드입니다.

Demo

텍스트에어리어 태그는 textarea 를 사용한다.

텍스트 필드 태그는 한줄 입력 / 텍스트에어리어 태그는 여러 줄 입력

텍스트에어리어 태그의 스타일을 지정할 때에 class="사이즈" 순서를 지키도록 한다.

모바일에서 텍스트 필드 태그의 width는 디폴트 값이 100% 이다.

1
<textarea class="dn_textarea_md" placeholder="기타 의견을 입력해주세요"></textarea>
cs

Textarea Sizes

텍스트에어리어의 사이즈를 간편하게 적용하기 위해 스타일을 지정하여 사용하세요.
모바일의 경우 width는 100%로 정해져 있어 높이 값을 기준으로 사이즈 구분이 되어있습니다.
텍스트에어리어의 사이즈는 .dn_textarea_sm, .dn_textarea_md, .dn_textarea_lg 로 구성 되어있습니다.

Small 사이즈

1
2
3
<textarea class="dn_textarea_sm" placeholder="기타 의견을 입력해주세요"></textarea>
<textarea class="dn_textarea_sm">.dn_textarea_sm</textarea>
<textarea class="dn_textarea_sm" disabled>.dn_textarea_sm</textarea>
cs

Textarea Disabled

텍스트에어리어를 비활성화 시키려면 disabled 를 추가하세요.

1
<textarea class="dn_textarea_md" disabled>disabled</textarea>
cs

Textarea Focus

텍스트에어리어가 focus 되었을 경우 하단과 같은 스타일이 적용됩니다.

1
<textarea class="dn_textarea_md">focus</textarea>
cs
